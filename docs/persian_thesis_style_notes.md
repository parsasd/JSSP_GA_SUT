# Persian Thesis Language Notes (CS / Scheduling)

This note captures terminology and prose conventions verified against Persian academic usage (Magiran, ViraScience, DOAJ-linked Persian papers) and applied to the thesis.

## Terminology decisions

- `Job Shop Scheduling Problem (JSSP)` -> `مسئله زمان‌بندی کارگاهی`
- `Dispatching Rules` -> `قواعد اولویت‌دهی`
- `Memetic Algorithms` -> `الگوریتم‌های ممتیک`
- `Multi-Armed Bandit` -> `مسئله راهزن چندبازویی`
- `Pareto-optimal` -> `پارتو-بهینه`
- `Trade-off` -> `موازنه`

## Tone conventions used

- Prefer concise formal prose over conversational structures.
- Prefer Persian academic connectors in moderation (`ازاین‌رو`, `بااین‌حال`, `درنتیجه`, `بدین‌ترتیب`).
- Keep core technical acronyms in English where standard in theses (`NSGA-II`, `MOEA/D`, `SPEA2`, `HV`, `IGD`).
- Keep equations, symbols, and references unchanged while improving sentence flow around them.

## Scope applied

- Front matter and all chapter files under `thesis-main/thesis/front/*.tex` and `thesis-main/thesis/chapters/*.tex`.
- Language-level normalization only (no structural/template changes).
