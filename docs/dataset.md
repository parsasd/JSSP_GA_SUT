# Dataset Notes

This project uses the canonical corpus file `data/raw/instance_data.txt` from the repository:
- [thomasWeise/jsspInstancesAndResults](https://github.com/thomasWeise/jsspInstancesAndResults)

The corpus aggregates widely-used benchmark families from OR-Library and Taillard:
- FT, LA, ORB, ABZ, TA (and others)

## Licensing and provenance
- Repository license text is vendored at `data/licenses/jsspInstancesAndResults_LICENSE.txt`.
- The dataset file itself includes provenance and citation notes in its header.

## Included subsets
- Quick subset: `ft06`, `la01`.
- Full subset (default): `ft06`, `la01`, `la06`, `la16`, `orb05`, `ta01`.

## Integrity
Checksums are generated by:

```bash
python scripts/prepare_data.py
```

Outputs:
- `data/manifests/raw_checksums.csv`
- `data/manifests/processed_checksums.csv`
